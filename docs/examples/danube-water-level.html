How this page was made

An agent asked ovf-data-mcp for the station, checked coverage, then ran three server-side aggregations — no scraping, no manual downloads:

vizugy stations search Budapest --watercourse Duna
vizugy observations aggregate surface:1026 --start 2025-07-18T00:00:00Z \
    --end 2026-07-18T00:00:00Z --interval daily --operation avg   # + min, max

Source: Országos Vízügyi Főigazgatóság (OVF) via the public VRAQuery API · station Budapest (1026), Duna, metric “Felszíni vízállás” (surface water level, cm), data type “operatív összefésült” · retrieved 2026-07-18 10:28 (UTC). Aggregation buckets follow upstream hydrological-day boundaries. Operational values may be preliminary; for checked data follow OVF’s formal data-request process.
